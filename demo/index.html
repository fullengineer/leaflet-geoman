Leaflet Polygon Management

An Open Source Leaflet Plugin for editing polygons, specifically made for Leaflet 1.0

Getting Started

Creating A Polygon

Click the Icon on the top left to start drawing mode. Then just click around on the map to create a polygon. Click on the first marker to finish drawing.
The 'pm:create' event will be fired after finishing the polygon.

How it works

To enable drawing of polygons, simply add the controls to the map with
L.PM.addControls(map2);

Editing A Polygon

Click on the polygon to toggle edit mode.
You can add new markers by clicking/dragging the transparent markers between the current ones.
Remove a marker with right click.

How it works

With polygon.pm.toggleEdit() you can toggle the edit mode for any polygon on the map.
Put it whereever you want. For example inside a click handler on a button or the polygon itself.

Editing A Layer Group

Same as above. The LayerGroup consists of Polygons and will just foward all events.
So when you toggle edit mode for a layerGroup, you're actually toggling edit mode for every
layer inside the group.

How it works

With layerGroup.pm.toggleEdit() you can toggle the edit mode for any layerGroup on the map.
